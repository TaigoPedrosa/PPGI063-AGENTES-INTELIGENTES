# DreamerV3 (Danijar Hafner)

---

## What it is

Model-based reinforcement learning algorithm that learns a world model from pixels and trains a policy entirely "in imagination", inside the learned model, without ever touching the real environment during policy training.

Published in **Nature 2025**. The third generation of the Dreamer line (V1: 2020 → V2: 2022 → V3: 2025), and the most mature open-source approach to training agents inside world models.

---

## Key concepts

- **World model as RSSM (Recurrent State-Space Model):** learns to predict observations, rewards, and episode continuations from actions
- **Training in imagination:** the policy (actor) and value function (critic) are trained on "dream" rollouts generated by the world model
- **Single set of hyperparameters:** works across Atari, DMC, Crafter, Minecraft without task-specific tuning
- **Symlog predictions:** symmetric logarithmic scaling for reward/value heads, enabling learning across different reward magnitudes

---

## Hardware requirements

- GPU recommended (JAX-based, uses GPU acceleration)
- Standard RTX-class GPUs sufficient for most environments
- Minecraft experiments require more compute (multiple GPUs)
- Lightweight environments (Crafter, DMC) run on a single consumer GPU

## Installation and training

```bash
git clone https://github.com/danijar/dreamerv3.git
cd dreamerv3
pip install -e .

# Train on Crafter (lightweight, good for testing)
python dreamerv3/main.py --configs crafter --run.train_ratio 32

# Train on Atari (e.g., Breakout)
python dreamerv3/main.py --configs atari --task atari_breakout

# Train on DeepMind Control Suite
python dreamerv3/main.py --configs dmc_vision --task dmc_walker_walk
```

## What you can do with it

1. **Train an agent from scratch on pixels**: no reward shaping, no pre-training, no task-specific hyperparameters
2. **Visualize imagined trajectories**: see what the world model "dreams" vs reality
3. **Test generalization**: single algorithm handles discrete (Atari) and continuous (DMC) control
4. **Understand model-based RL**: the cleanest instantiation of the "agents dreaming" thesis

---

## Key results

- **Minecraft diamond:** first algorithm to obtain diamond from scratch purely from pixel observations via imagination training
- **Atari:** surpasses human-level on most games with a single set of hyperparameters
- **DMC:** competitive with specialized methods
- **Crafter:** strong multi-objective performance (survival + crafting + exploration)

## Difference from JEPA-based models

| Aspect | Dreamer | JEPA (V-JEPA 2, LeWM) |
|---|---|---|
| Training signal | Reward from environment | Self-supervised (no reward) |
| Planning | Policy trained in imagination | MPC with CEM at test time |
| Prediction space | Pixel reconstruction + reward | Latent embeddings only |
| Goal specification | Reward function | Goal image |
| Deployment | Learned policy executed directly | Re-plans at every step |

Both are "world models" but with fundamentally different planning philosophies. Dreamer learns a *policy* inside the world model; JEPA uses the world model as a *simulator* for online planning.
