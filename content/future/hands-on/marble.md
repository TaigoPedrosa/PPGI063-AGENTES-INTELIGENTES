# World Labs: Marble

---

## What it is

Generative multimodal world model that creates **persistent, navigable 3D worlds** from text, images, video, or coarse 3D layouts. Founded by **Fei-Fei Li** (Stanford, ImageNet creator).

The most accessible way to experience "world generation", runs entirely in a **browser**, no code or GPU needed. Also has a developer API.

---

## Key concepts

- **Spatial intelligence:** AI that understands and generates 3D space, not just 2D images
- **Persistent worlds:** generated environments are consistent, you can explore them indefinitely without morphing or inconsistency
- **Multimodal input:** create worlds from text, single image, multiple images, video, or 3D structures
- **Gaussian splatting:** 3D representation technique used for rendering (exportable)

---

## Hardware requirements

- **Web app:** any modern browser (desktop recommended)
- **API:** no GPU needed on your side, generation runs on World Labs servers
- **VR viewing:** compatible VR headset (optional)

---

## Access: Web app (free)

1. Go to https://marble.worldlabs.ai
2. Sign up with email
3. Start creating worlds

### Features

| Feature | Description |
|---|---|
| **Gallery** | Browse community-created worlds for inspiration |
| **Create** | Generate new 3D worlds from text, image, video, or 3D structure |
| **Studio** | Edit, expand, combine worlds; record cinematic walkthroughs |
| **Export** | Download as Gaussian splats, meshes, or videos |

### Input types
- **Text prompt:** "A futuristic robotics lab with mechanical arms and holographic displays"
- **Single image:** upload a photo, get a full 3D environment based on it
- **Video:** short clips (under 100 frames) converted to navigable 3D
- **3D structure:** coarse layout → detailed world

---

## Access: World API (developers)

Launched January 2026. Programmatic access to generate 3D worlds.

```
POST /v1/worlds
{
  "prompt": "A futuristic robotics lab with mechanical arms",
  "input_type": "text"
}
```

- Sign up at https://docs.worldlabs.ai
- Generate API keys via the World Labs Platform
- Pricing: credit-based (see platform for current rates)
- Async generation: submit request → receive navigable 3D world

| Video recordings | Cinematic captures from the Studio |

---

## Why this matters for world models

Marble demonstrates a different facet of world models than JEPA or Dreamer:
- **Not about planning or control**: it's about *generating* and *understanding* 3D space
- Shows that "world models" is broader than just robotics
- Spatial intelligence is a building block for agents that need to understand physical environments
- The World API enables building applications where AI generates environments on demand
